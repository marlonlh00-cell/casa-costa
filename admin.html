  2. Inside the project → Click web icon (</>) → Register app → Copy the firebaseConfig object → Paste it in admin.html AND index.html replacing the placeholder text
  3. In Firebase left menu → Firestore Database → Create database → Start in test mode → Done
  4. Go to emailjs.com → Create account → Add Email Service → Connect Gmail (laramarlon996@gmail.com) → Note the Service ID
  5. In EmailJS → Email Templates → Create template → Set Subject to "You're confirmed. Casa Costa 🌊" → Paste the HTML body below → Note the Template ID
  6. In EmailJS → Account → API Keys → Copy your Public Key
  7. Paste all three EmailJS values into admin.html where it says REPLACE_EMAILJS
  8. Once everything is pasted, delete this banner or set id="setupBanner" to class="hidden"
